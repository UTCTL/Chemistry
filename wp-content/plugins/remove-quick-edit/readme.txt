== Description ==

This plugin removes the "quick edit" button located in the pages and posts list by CSS. Ideal to use with qTranslate

== Installation ==

Just donwload the zip file, upload it to your wordpress via install plugins page (or uncompress it to your wordpress wp-content/plugins/ folder) and activate it. Once activated the "quick edit" button will automatically dissapear.

== Usage ==

Just activate the pluggin and the "quick edit" button will automatically dissapear in "edit pages" and "edit posts" sections of the admin panel.
